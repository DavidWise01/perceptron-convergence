PERCEPTRON CONVERGENCE — perceptron theory I of III (David: 'strong, middling, beginning'), the STRONG one. THE PROVEN PROMISE: if data is linearly separable with margin γ and radius R, the perceptron makes at most (R/γ)² mistakes before it stops — exact, dimension-free, independent of how many points (Novikoff & Block 1962). LIVE: separable points with a margin slider; the perceptron updates only on mistakes (w←w+y·x), the decision line rotates in, and the mistake count stays ≤ ⌈(R/γ)²⌉ — shrink γ and the bound balloons but the count never breaks it (verified in node: margins 0.6/0.3/0.15 → 1/1/9 mistakes vs bounds 16/56/262). VERDICT = PROVEN THEOREM (bedrock); the proof is two inequalities. Honest catch (still proven): only for SEPARABLE data — on non-separable it never converges; and it finds A separator, not the BEST one (that's max-margin/SVM).
